# OpenClaw

OpenClaw ist eine lokale, agentische Assistenzplattform, mit der du einen KI-Assistenten über verschiedene Kanäle (z. B. Chat, Cron-Jobs, Browser-Relay) steuern kannst.

## Was OpenClaw kann

- **Chat-Assistenz**: Gespräche in deinen angebundenen Kanälen führen
- **Automatisierung**: Wiederkehrende Aufgaben über Heartbeats und Cron-Jobs ausführen
- **Tool-Nutzung**: Dateien bearbeiten, Shell-Befehle ausführen, Web-Recherche durchführen und Browser automatisieren
- **Kontext & Gedächtnis**: Arbeitskontext und Notizen im Workspace verwalten

## Typischer Einsatz

OpenClaw eignet sich für persönliche Produktivität, technische Unterstützung und autonome Routineaufgaben in deinem eigenen Setup.

## Wichtig

OpenClaw läuft in deiner Umgebung und folgt deinen Freigaben und Sicherheitsregeln.
